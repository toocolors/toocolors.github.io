The OSI Model


How is data sent from a computer, through the internet, to another computer? Today, the internet uses the OSI Model.

The OSI Model has seven layers:

  1. Physical Layer
  2. Data Link Layer
  3. Network Layer
  4. Transport Layer
  5. Session Layer
  6. Presentation Layer
  7. Application Layer
Each layer helps data move across the internet, and relies on the layer below it to function.


The Physical Layer

The Physical Layer is responsible for sending and receiving individual bits.
When it receives a bit, it forwards it to the Data Link Layer.

The Data Link Layer

The Data Link Layer ensures that data is transfered from one node to another without errors.

The Network Layer

The Network Layer helps transmit data between two hosts that are located in different networks.
It is in charge of routing data inbetween different hosts and networks.
Devices such as routers and switches are commonly used to implement the Network Layer.

The Transport Layer

The Transport Layer ensures communication between two different hosts.
It further helps ensure that data is transfered to the correct application, on the correct host.
It also features extra features, such as flow control, which helps keep data flowing smoothly.
Common protocols in this layer include TCP and UDP.

The Session Layer

The Session Layer helps manage communication sessions between two hosts.

The Presentation Layer

The Presentation Layer manipulates data from the Application Layer to ready it for network transmission.

The Application Layer

The Application Layer functions as a point of access for applications to access the network.
