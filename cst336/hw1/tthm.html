Time to Hello World!


"Hello World" is a common term heard in programming. "Hello World" world programs seek to print the phrase in the least number of lines possible.

The number of lines needed varies between different programming languages. Some need only one or two lines, while others need several. The time it takes to print "Hello World" in any given language is known as "Time to Hello World" (TTHW).

This page is going to showcase the "Hello World" programs of different popular programming languages, and then compare how many lines it took for each language.


Python

print("Hello World")

Java

class HelloWorld {
 public static void main(String args[]) {
  System.out.println("Hello World");
 }
}

C

#include <stdio.h>
int main() {
 printf("Hello World");
}

C++

#include <iostream>
using namespace std;
int main() {
 cout << "Hello World";
}

C#

using System;
namespace HelloWorld {
 static void Main(string[] args) {
  Console.WriteLine("Hello World");
  Console.ReadKey();
 }
}

JavaScript

<script>
console.log('Hello World');
</script>
