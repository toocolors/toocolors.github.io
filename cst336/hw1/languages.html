In addition to their type, each programming language specializes in one or more tasks. They may focus on broad concepts such as simplicity or speed, or they may focus on specific fields such as data science.

This page will briefly cover some popular languages, their goals, advantages and disadvantages, and some of their notable characteristics.

Some languages this page will cover include:

  • Python
  • Java
  • SQL

Python

Python is a high-level, interpreted language that is known for being readable and easy to learn. It features extenstive library support, as well as a large and active community. It is commonly used for AI, data science, and web development.

Python is known for executing slowly, due to it being an interpreted language.

Python is notable for its easy to read syntax, which often features symbol English. It also omits semi-colons in favor of using white-space to designate pieces of code.

Java

Java is an object-oriented language. It is known for being easy to port to other hardware, which allows it to be used in a wide variety of environments.

Java is known for being verbose, which can make writing programs and maintenance challenging. It is also known for its slow startup times.

Instead of compiling to specific hardware, Java uses the Java Virtual Machine (JVM), which is able to run compiled Java code on a variety of hardware.

C

C is a compiled language that is known for its fast execution. It is used in operating systems, embedded systems, game engines, and many other programs that require efficient execution.

C lacks many modern features that other languages have. This makes errors and mistakes more easy to occur in C.

C is often called the mother of all programming languages as many other languages have borrowed syntax or features from it.

SQL

SQL is a specialized language designed managing databases. Similar to Python, it features easy to read syntax that makes it easy to learn. It is used anywhere where databases are used, making it common in many areas.

Because SQL is so specialized, it is incapable of performing many tasks. Integration with other programming languages is required.

HTML

HTML is a mark-up language used to structure content on web pages. The majority of websites rely on HTML in combination with CSS and JavaScript. Because of this, HTML is a very common and popular language. It is also used in moble app development.

HTML doesn't support logic, interactivity, dynamic content, or advanced styles on its own. It relies on other languages, namely JavaScript and CSS, to deliver these services.

JavaScript

JavaScript is a high-level language, interpreted language used for designing web applications. It is able to integrate with HTML to make robust, interactable websites and web applications.

To ensure compatability with older web browsers, servers, and websites, JavaScript retains several bugs and other quirks.

Despite its name, JavaScript has very in little in common with Java.
